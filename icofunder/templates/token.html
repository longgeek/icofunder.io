What's IOF?

{% trans "什么是 IOF Token？" %}

{% SETTINGS "SITE_DOMAIN" as site_domain %}

{% blocktrans %} {{ site_domain }} 将发行总量为 2 亿 的 {{ site_domain }} Coin，简称 IOF。该代币是基于以太坊智能合约 ERC-20 标准的代币，{{ site_domain }} 团队将承诺永不增发。{% endblocktrans %}

{% trans "IOF Token 有什么用？" %}

{% blocktrans %}IOF 将会成为 {{ site_domain }} 换币系统、去中心化交易所的基准货币。可以参与 ICO 众筹、抵扣手续费、去中心化交易所燃料、商家保证金、创新企业孵化基金等，持有 IOF 代币的用户将会不定期赠送去中心化交易所上线的新币，同时将永久享有每日利润的 20% 分红权。" {% endblocktrans %}

{% trans "IOF Token 的长期价值体现？" %}

{% blocktrans %}为保证 IOF Token 价格增长，{{ site_domain }} 团队将会在交易所上线后，用每个季度净利润的 50% 回购 、销毁 IOF Token。回购的 IOF 直接销毁，回购记录将会第一时间公布，用户可通过区块链浏览器查询，确保公开透明，直至销毁到总量为 1 亿个 IOF 币为止。{% endblocktrans %}

{% trans "什么是换币系统？" %}

{% blocktrans %}传统的币币交易往往有很多局限，{{ site_domain }} 将提供根据当前虚拟货币按市场最优汇率来换取其它代币。{% endblocktrans %}

{% trans "什么是去中心化交易所？" %}

{% blocktrans %}{{ site_domain }} 团队将在 EOS 正式落地后，开发基于 EOS 的去中心化交易所。IOF 代币也将是去中心化交易所的燃料。{% endblocktrans %}
